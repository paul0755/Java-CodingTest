1. 🔥 불과 사람 BFS를 같이 섞음

처음에는 불(*)과 사람(@)을 같은 큐에 넣고 bfs를 돌림.

→ 이렇게 하면 불과 사람의 진행 순서가 뒤죽박죽 돼서 문제 조건(불 → 사람 순서 유지)이 깨짐.

3. 🌀 BFS 순서 제어 실패

bfs(fireQ); bfs(personQ); 이렇게 따로 호출.

→ 이렇게 하면 불이 전부 퍼진 후 사람 이동이 돼서 실제 시뮬레이션과 다름.
